## UPI Behavioural analysis in India
This project performs time series exploratory data analysis on UPI transaction datasets to understand user behavioural patterns in India's digital payment ecosystem

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook

## Dataset
Official NPCI monthly statistics --> npci.org.in April 2019 to May 2026 - 78 months.

## Objective
To analyse digital payment behaviour trends and extract trends from UPI transactions patterns.
